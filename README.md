# Three.js FPS Demo

A first-person shooter demo built with Three.js featuring:
- Octree-based collision detection
- Static triangle mesh collisions
- Character capsule collision
- First-person camera controls (WASD + mouse)
- Jump and ball throwing mechanics

## Usage

Open `index.html` in a modern browser.

### Controls
- **WASD** - Move
- **Mouse** - Look around
- **Space** - Jump
- **Click** - Throw ball
